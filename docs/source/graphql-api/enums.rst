Enums
=====

The ``gpp_client.generated.enums`` module contains enums generated from the
GraphQL schema.

Use these enums when constructing typed input models or comparing returned
values.

Example
-------

.. code-block:: python

   from gpp_client.generated.enums import ObservationWorkflowState
